# рҹҺ® Welcome to Terminal Arcade Hub

Bring the golden age of arcade gaming back to your terminal.

This project is a collection of **classic retro games** recreated entirely in **Python**, all accessible through an interactive terminal launcher inspired by old-school arcade machines.

Whether you want a quick game of **Snake**, survive waves in **Space Invaders**, or stack blocks in **Tetris**, everything is just one key press away.
